Confidence-Guided Vibration-to-Acoustic Distillation for Bottle Classification in Compactors

Authors:
Junfu Zhang, Yongxu Ren, Felix Deichsel, Philipp Beckerle

Project Overview

This repository contains the source code, dataset, trained models, and experimental results used in the study:

Confidence-Guided Vibration-to-Acoustic Distillation for Bottle Classification in Compactors.

The project investigates cross-modal knowledge distillation from vibration signals to audio signals for bottle classification in reverse-vending and compaction systems. During training, a vibration-based teacher model transfers knowledge to an audio-only student model. During inference, only audio input is required.

Repository Contents

  • Source code for teacher, student, and distillation models
  • Audio and vibration datasets
  • Processed training data
  • Trained model checkpoints
  • Alpha-sweep experiments
  • Noise robustness evaluation results
  • Confusion matrices and classification reports
